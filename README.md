To run this Fake Twitter project you need following Angular and Node version -

Angular CLI: 16.0.3
Node: 18.16.0

To run the project you need to execute the following command -

cd fake-twitter
npm install
ng build
ng serve

Then browse the http://localhost:4200/ in chrome
